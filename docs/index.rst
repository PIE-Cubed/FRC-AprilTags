FRC-AprilTags's Documentation
=============================

Welcome! FRC-AprilTags is a Python package developed by FIRST Robotics Competition (FRC) team 2199, The Robo-Lions,
that aims to simplify the detection of AprilTags for all teams in the coming FRC seasons.
This package utilizes the `RobotPy <https://pypi.org/project/robotpy/>`_,
`Pupil Apriltags <https://pypi.org/project/pupil-apriltags/>`_, and `OpenCV <https://pypi.org/project/opencv-contrib-python/>`_
libraries to make this detection of AprilTags as simple and straightforward as possible.

Check out the :ref:`getting started <starting>` section to begin learning about this package!
