FRC-AprilTags's Documentation
=============================

Welcome! RobotPy is a project created by a community of FIRST mentors and students dedicated
to developing python-related projects for the FIRST Robotics Competition.
This documentation site contains information about various projects that RobotPy supports,
including guides and API references.

**FRC-AprilTags** is a Python library for FIRST Robotics Competition (FRC) teams
that aims to simplify the detection of AprilTags in the coming FRC season.
It utilizes the `RobotPy <https://world.openfoodfacts.org/>`_, `pupil-apriltags <https://world.openfoodfacts.org/>`_, `pupil-apriltags <https://world.openfoodfacts.org/>`_ libraries
and offers a simple and straightforward way to detect and utilize the data.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
